# 脖子以下乳液保養
- 乳液：一天擦純雅漾Avenue，交替一天擦杏貝他健混合雅漾
- 混藥比例：雅漾3 杏貝他健1（杏貝他健約一元硬幣大小，然後雅漾按到底3下）
- 頻率：早晚各一次，除了早晚抹藥，如果時間許可，中間可以在補雅漾
![[IMG_1526 1.jpeg]]

# 臉部乳液保養
- 只擦雅漾即可，不需要混藥
- 頻率：跟擦身體乳液一起

# 頭部乳液保養
- 如果頭皮搔癢，可以用淨膚妥混合水1:4噴於頭上
- 如果沒有症狀則不需使用
- 已混合好放在冰箱門上層
![[IMG_1528.jpeg]]

# 洗澡
頻率：每天一次
將入浴劑挖滿紅色小盒子，之後用80-90攝氏度的熱水溶解，溶解好的入浴劑呈現乳白色，再加入冷水讓洗澡水溫度達到約30攝氏度（大約90華氏度）
