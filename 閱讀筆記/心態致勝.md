# 成長心態與定型心態
[[成長心態]]的人喜歡挑戰，所以遇到挫折或是困難時，會表現出積極努力的態度
[[定型心態]]的人則相信人們的能力是固定的，他們也許在某個領域上非常成功，但是一但遭遇到困難或是挫折，則可能開始自暴自棄，認為自己能力不夠

# 社交能力也是成長心態
通常羞怯的人（I人）是定型心態居多，但也有I人是成長心態的，這兩種人區別在於，成長心態的人把社交視為一種挑戰，因此就算在社交場合的一開始表現害羞，但再過幾分鐘內通常會表現得較為外放
I人但擁有定型心態的人則是相反，可能會認為自己社交能力不佳，所以越來越封閉自我

# 讚美努力而不是讚美成果
如果要對孩子讚美，盡量避免讚美他們的成果或是聰明才智，這樣容易使他們養成定型心態，導致他們不願成長、不喜歡挑戰、害怕失敗

❌ 錯誤示範
「你這次的數學考試考得很棒，你真聰明」

✅ 正確示範
「你這道數學題目嘗試了不同的算法，終於搞懂了，你想了很多的做法，終於找到了可行的方法，我喜歡你這種努力的方式」
