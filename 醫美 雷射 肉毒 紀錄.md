# **術後洗臉**

一個禮拜內不建議用有去角質的洗面乳

# **防曬**

- 男生建議白天擦防曬就好，不然可能會太油，晚上再擦乳液保濕
- 防曬要用SPF50的，然後擠50元硬幣大小或是兩個指節大小
- 術後建議要**物理防曬** 外出要戴帽子

# 蜂巢皮秒雷射施打過程
美容師會先幫你卸妝，接著敷麻藥大約20分鐘，卸掉麻藥之後，會移動到機器的地方開始施打，前半段是普通的皮秒雷射，搭配麻藥打起來其實沒什麼感覺，如果不是因為燒焦味跟聲音甚至沒有感受到在打，而中途加強能量後則會略有刺痛感，到最後換成蜂巢的探頭後，刺痛感會明顯增強

# 肉毒施打過程
醫生會先請你抬眉跟皺眉，觀察一下紋路，接著拿筆做出記號，接著會讓你閉上眼睛然後醫生就開始注射肉毒，過程就像是打針一樣，只不過針筒會插比較淺然後打下去的瞬間會有點漲漲的，抬頭紋從左到右應該打了7-9個點（沒有細算），而眉頭則有3-4個點，比較靠邊緣的點不知道是因為麻藥不夠的緣故，會比較痛，而眉頭則是每一針都偏痛

注意事項
- 效果會慢慢出來，兩個禮拜左右效果完全出來
- 會慢慢感受到皺眉跟抬頭有點沒力
- 不要去太熱的地方例如溫泉、蒸氣室、桑拿室等，避免無效
- 不要劇烈運動
- 可以維持4-6個月
