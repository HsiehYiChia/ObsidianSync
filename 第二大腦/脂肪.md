## 脂肪是什麼？

脂肪（Fat）主要是由 **脂肪酸（fatty acids）** 組成的。  
而「脂肪酸」的化學結構可以用**碳鏈長度**與**是否含有雙鍵（＝不飽和鍵）**來分類。

---

## 🧬 按「結構」分類

| 類型                             | 結構特徵          | 常見狀態 | 主要來源               | 對健康的影響          |
| ------------------------------ | ------------- | ---- | ------------------ | --------------- |
| **[[飽和脂肪]]（Saturated Fat）**    | 沒有雙鍵，碳鏈被氫「飽和」 | 固體居多 | 奶油、豬油、牛油、椰子油       | 過量可能升高壞膽固醇（LDL） |
| **[[不飽和脂肪]]（Unsaturated Fat）** | 含有一個或多個雙鍵     | 液體居多 | [[橄欖油]]、魚油、[[酪梨油]] | 有助心血管健康         |
| **[[反式脂肪]]（Trans Fat）**        | 不飽和但經氫化處理變硬   | 固體   | 人造奶油、餅乾、速食炸物       | 增加              |



## 🩺 整體健康建議

| 類型                | 盡量減少              | 適量攝取 | 可多攝取    |
| ----------------- | ----------------- | ---- | ------- |
| 飽和脂肪              | ✅ 控制（不超過總熱量的 10%） |      |         |
| 反式脂肪              | 🚫 避免             |      |         |
| 單元不飽和脂肪           |                   | ✅ 適量 |         |
| 多元不飽和脂肪 (Omega-3) |                   |      | ✅ 建議多攝取 |

---

## 🔍 一句話整理：

> 飽和脂肪讓油變「硬」，不飽和脂肪讓油變「軟」，反式脂肪是「硬化失敗版」，對身體最傷。



## 🌱 那反式脂肪會自然存在嗎？
✅ **答案：少量會自然存在，但大多是人工產生。**

### 1️⃣ **人工反式脂肪（主要來源）**
- 在工廠製造時，植物油經「**部分氫化（partial hydrogenation）**」變硬，用來：
    - 提高耐放性
    - 增加酥脆口感
    - 降低成本
- 常見於：
    - 人造奶油（margarine）
    - 烘焙食品（餅乾、派皮、酥皮）
    - 速食炸物（重複高溫油炸的油）

👉 這是**最有害的形式**，現今多國已禁止或限制。

---

### 2️⃣ **天然反式脂肪（微量存在）**
- 部分反芻動物（牛、羊）腸道中細菌會自然產生少量反式脂肪（例如 **共軛亞油酸 CLA**）。
- 因此在：
    - 牛奶
    - 牛肉、羊肉  
        中會檢出極少量天然反式脂肪。

🔹這種天然反式脂肪含量極低、結構稍不同，**目前沒有明確證據顯示會造成同等危害**。  
換句話說，**自然肉奶中的反式脂肪不需特別避免，但人工氫化油一定要避開。**

---

## ✅ 小結一句話：
> - **飽和脂肪**：可吃但別太多（控制量）
> - **不飽和脂肪**：健康脂肪，多吃好油（橄欖油、魚油）
> - **反式脂肪**：工業製造最危險，盡量「0」攝取
